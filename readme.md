## Run Locally

Clone the project

```bash
  git clone https://github.com/MohitSaraff/Haunted_House.git
```

Go to the project directory

```bash
  cd Haunted_House
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
